## Локальный запуск
1. Склонировать репозиторий
2. Установить отсутствющие модули npm
    ```
        npm install
    ```
3. Запустить локальный сервер
    ```
        npm run start
    ```
4. Запустить локальный сервер с хот-релоудем
    ```
        npm run dev
    ```
